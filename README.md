<h1>Trigonometrijos pagrindai</h1>

<div class="box">
<h2>Sąvokos</h2>
<p><b>Trigonometrija</b> – matematikos sritis, nagrinėjanti kampų ir trikampių kraštinių ryšius.</p>
<p><b>Įžambinė</b> – ilgiausia kraštinė.</p>
<p><b>Priešpriešinė</b> – kraštinė prieš kampą.</p>
<p><b>Prigludusioji</b> – kraštinė prie kampo.</p>
</div>

<div class="box">
<h2>Pagrindinės formulės</h2>
<p><b>sin(α)</b> = priešpriešinė / įžambinė</p>
<p><b>cos(α)</b> = prigludusioji / įžambinė</p>
<p><b>tan(α)</b> = priešpriešinė / prigludusioji</p>
</div>

<div class="box">
<h2>Savybės</h2>
<p>sin²(α) + cos²(α) = 1</p>
<p>tan(α) = sin(α) / cos(α)</p>
<p>sin ∈ [-1; 1]</p>
<p>cos ∈ [-1; 1]</p>
</div>
